Module imaplib_ssl_fix

source code

Work-around for Python Issue 5943 (http://bugs.python.org/issue5949).

This will patch imaplib's IMAP4_SSL.readline method with the fixed verion in Python versions that are known to have the problem.

The problem definitely exists in Python 2.5 and 2.6 up until but not including 2.6.5. It was also fixed in Python 2.7 alpha 2 so no attempt is made to patch 2.7 versions.

Please let me know if there's more Python versions that should be patched.

Efforts are made to only perform the patch once.
