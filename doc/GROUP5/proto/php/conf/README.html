Introduction

For more information, see the gus doc

For obvious reasons, this conf directory should not be web readable. The best practice at this stage of GUS is to soft link the index.php file in the main source directory where you like in your main web directory. If you are in a development environment, you can simple password protect this source directory from common use.

Configure

Edit the main.php file of this directory to match the setup you want.

  • DBMODE - this value can be 1 of 3 values
    • mysql
    • ldap (not yet implemented)
    • file (not yet implemented)
  • DBHOST,DBADMIN,DBPW - applicable only to mysql and ldap DBMODES

Install

  1. Configure main.php, as mentioned in the Configure section above, of course.
  2. Create the DBADMIN user you specified in main.php in your actuall database . Do not use your mysql root user.
      i.e.:
    • mysql:
      • mysql -u root -p
      • mysql: CREATE USER 'gusadm'@'localhost' IDENTIFIED BY 'password';
      • mysql: grant all privileges on *.* to 'gusadm'@'localhost';
  3. Navigate to install.php.
  4. Done!

Uninstall

Navigate to uninstall.php. Done!
